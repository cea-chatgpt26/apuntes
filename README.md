## Estructura de cada archivo

Todos los días siguen el mismo patrón:

1. **Recapitulando** — pregunta rápida sobre el día anterior
2. **Contenido** — teoría en bloques cortos con ejemplos
3. **Vídeo** — referencia de YouTube con duración
4. **Ejercicios** — entre 2 y 3, con respuestas desplegables donde aplica
5. **Trabajo en equipo** — 30 min, dinámico y conectado al contenido del día
6. **Lo que aprendiste hoy** — resumen en 5 puntos
